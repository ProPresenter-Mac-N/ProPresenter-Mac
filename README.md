## Overview

ProPresenter Professional Suite is a complete live production platform with NDI output, stage display, and MIDI integration for professional event production.

NDI output sends video feeds over the network for integration with broadcast production systems and video switchers.

Stage display provides customizable confidence monitors showing next-slide, notes, timers, and current-slide for presenters and musicians.

MIDI and MIDI-over-network integration triggers presentation cues from lighting boards, audio consoles, and other production equipment.

ProPresenter Professional Suite is the live presentation platform for professional production teams that need multi-screen, broadcast-ready presentation with the integration and reliability that live event production demands.

## Performance & Integration

ProPresenter is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system.

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
